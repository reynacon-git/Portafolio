# Acerca de mi 
Mi formación en Psicología, me permite ofrecer una perspectiva única en el análisis de datos. Me enfoco en comprender a profundidad las necesidades y preferencias de los consumidores para diseñar soluciones de datos que generen un verdadero impacto en el negocio. Mi objetivo es construir puentes entre los datos y las personas.
